  ## Description 
  The Tempurah weather app is a React app that utilizes Node.js, Express and the openWeather API to look up weather conditions for its users. The streamlined UI is great for minimalists that need quick information to go about their day. Per request, the app also displays the mean, median and mode of day temperatures for the next three days.

  ## Installation
    
    1. To use this code for your own project, download the Tempura repository locally to your computer.
    2. Navigate to the Tempura directory on your terminal
    2. Run npm install in your bash/command line to install dependencies listed in the 'package.json' file.

  ## Usage
      
    1. To run application, invoke with npm start in your bash/command line. 
    2. View the live application in your browser at localhost:3000
